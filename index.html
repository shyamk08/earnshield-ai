Process

Zero friction,
full protection.

Six steps from registration to payout — all automated, all instant.

01
📝
Register & Onboard
Worker registers with location, platform (Swiggy/Zomato), and work details. KYC verification via Aadhaar API.
02
🤖
AI Risk Assessment
ML model evaluates environmental, geographic, and historical disruption data to generate a risk score for the worker's area.
03
💸
Weekly Premium Set
Dynamic pricing engine generates a personalized weekly premium. Low-risk zones get discounts; high-risk zones have adjustments.
04
📡
Real-Time Monitoring
Weather APIs, AQI feeds, and GPS track conditions every 5 minutes. The system watches all parametric thresholds continuously.
05
⚡
Disruption Detected
When a trigger threshold is crossed, the AI engine validates the event against fraud patterns and worker location data instantly.
06
✅
Instant Payout
Claim auto-triggers. Razorpay/Stripe processes the payout to the worker's UPI or bank account in under 2 seconds.
Parametric Triggers

When conditions break,
payouts kick in.

Claims are fully automatic — no manual filing, no adjuster. The system decides based on objective data.

🌧️
Heavy Rainfall
When rainfall intensity exceeds the threshold in a worker's GPS zone, deliveries are impossible. Claim triggers automatically for all affected workers in that zone.
Threshold: >10mm/hour
🌡️
Extreme Heat
Riding in extreme temperatures poses serious health risks. When ambient temperature breaches the safety threshold, the platform compensates for work stoppage.
Threshold: >42°C for 2+ hours
😷
Poor Air Quality
Severe air pollution makes outdoor work dangerous. AQI data from monitoring stations triggers income protection when pollution spikes dangerously.
Threshold: AQI >200 (Very Poor)
🚫
Zone Restrictions
Government-declared curfews, lockdowns, or restricted movement zones automatically trigger protection for workers whose areas are affected by official orders.
Threshold: Official order issued

Technology

Built on a production-
grade tech stack.

Every layer purpose-built for reliability, real-time performance, and scale.

Frontend
React.js + Tailwind CSS
Responsive PWA-ready interfaces for workers and admins with real-time data updates.
Backend
Node.js + Express.js
High-throughput API server handling thousands of concurrent claim evaluations per second.
Database
Firebase + MongoDB
Real-time database for live triggers; MongoDB for worker profiles, claims, and audit logs.
AI / ML
Python + Scikit-learn
Risk prediction models, dynamic pricing engine, and anomaly detection for fraud prevention.
Data Feeds
Weather + Maps APIs
OpenWeatherMap, AQICN, and Google Maps APIs feeding 5-minute refresh condition data.
Payments
Razorpay / Stripe
UPI-first instant payout engine with sandbox testing. Sub-2 second disbursement to workers.
